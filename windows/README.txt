= YDCMD =
Актуальный исходный код и документация находятся по ссылке:
https://github.com/abbat/ydcmd

Для работы под Windows вам может потребоваться обновить список сертификатов
доверенных центров сертификации (ca-root.crt), который можно найти по ссылке:
http://curl.haxx.se/docs/caextract.html

Перед началом работы рекомендуется создать файл конфигурации, который должен
находиться по пути C:\Users\<Имя Вашего Пользователя>\.ydcmd.cfg

На некоторых системах для работы программы может потребоваться установить
Microsoft Visual C++ Redistributable Package, который можно скачать по ссылке:
https://support.microsoft.com/en-us/help/2977003/the-latest-supported-visual-c-downloads


= YDCMD =
The actual source code and documentation can be found at:
https://github.com/abbat/ydcmd

For Windows you may need to update the list of trusted certification
authorities (ca-root.crt), which can be found at:
http://curl.haxx.se/docs/caextract.html

Before you begin it is recommended to create a configuration file:
C:\Users\<Your User Name>\.ydcmd.cfg

On some systems, the program may require to install Microsoft Visual C++
Redistributable Package which can be found at:
https://support.microsoft.com/en-us/help/2977003/the-latest-supported-visual-c-downloads
